## 编程助手服务端

Server文件下是vscode编程助手服务端代码文件

### 安装

用户可以尝试以下步骤进行启动

```sh
# 1.进入Server目录
cd Server
# 2.设置启动虚拟环境
conda create -n XXX python=3.10
conda activate XXX
# 3.安装项目依赖
pip install -r requirements.txt
# 4.启动项目
python Listen.py
```
